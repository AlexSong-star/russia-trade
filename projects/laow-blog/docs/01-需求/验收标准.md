# 老六博客 - 验收标准

> 📝 博客定位：老六的个人技术博客，展示老六的思想、个性、本事  
> 内容创作：由老六主导，每天 ≥ 2 篇  
> 发表流程：创作 → 同步强哥 → 同意后发布

## 交付标准

### P0 核心功能

| 功能 | 验收条件 |
|------|----------|
| 首页文章列表 | 打开首页能看到文章列表，分页正常 |
| 文章详情页 | 点击文章能进入详情页，内容完整显示 |
| Markdown渲染 | 代码块高亮、图片正常显示 |
| 分类页面 | 点击分类能看到该分类下的文章 |
| 标签页面 | 点击标签能看到带该标签的文章 |
| 后台管理 | 能通过 /admin 或类似路径进入管理页 |
| 文章管理 | 能在后台新增、编辑、删除文章 |

### P1 重要功能

| 功能 | 验收条件 |
|------|----------|
| 点赞功能 | 点击点赞按钮，数字+1 |
| 阅读量统计 | 打开文章详情，阅读量+1 |
| 评论功能 | 能发表评论，提交后显示在评论区 |
| 评论管理 | 能在后台看到评论列表，可删除 |
| 视频嵌入 | B站视频能正常播放 |

### P2 增强功能

| 功能 | 验收条件 |
|------|----------|
| 暗黑模式 | 切换模式后页面配色改变 |
| 搜索功能 | 输入关键词能搜到相关文章 |

## 视觉验收

- [ ] 页面风格炫酷，有特色
- [ ] 手机端浏览正常
- [ ] 图片加载正常
- [ ] 视频播放正常
- [ ] 动画效果流畅

## 性能验收

- [ ] 首页加载时间 < 3秒
- [ ] 文章详情页加载时间 < 3秒
- [ ] 无明显卡顿
