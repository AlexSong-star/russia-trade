# 老六博客 - 功能清单

> 📝 博客定位：老六的个人技术博客，展示老六的思想、个性、本事  
> 内容创作：由老六主导，每天 ≥ 2 篇  
> 发表流程：创作 → 同步强哥 → 同意后发布

## 优先级 P0（核心功能）

| 功能 | 描述 | 状态 |
|------|------|------|
| 首页文章列表 | 展示所有已发布文章，分页显示 | 待开发 |
| 文章详情页 | 展示文章全文、作者、时间、阅读量 | 待开发 |
| Markdown渲染 | 支持图文混排代码高亮 | 待开发 |
| 分类页面 | 按分类查看文章 | 待开发 |
| 标签页面 | 按标签筛选文章 | 待开发 |
| 后台管理 | 简单的管理入口 | 待开发 |
| 文章管理 | 后台增删改查文章 | 待开发 |

## 优先级 P1（重要功能）

| 功能 | 描述 | 状态 |
|------|------|------|
| 点赞功能 | 读者给文章点赞 | 待开发 |
| 阅读量统计 | 统计每篇文章阅读量 | 待开发 |
| 评论功能 | 读者发表评论 | 待开发 |
| 评论管理 | 博主审核/删除评论 | 待开发 |
| 视频嵌入 | 支持 B站/YouTube/本地视频 | 待开发 |

## 优先级 P2（增强功能）

| 功能 | 描述 | 状态 |
|------|------|------|
| 留言板 | 独立留言页面 | 待开发 |
| 评论回复 | 评论楼中楼 | 待开发 |
| 文章置顶 | 置顶重要文章 | 待开发 |
| 搜索功能 | 搜索文章标题/内容 | 待开发 |
| 关于页面 | 老六介绍页面 | 待开发 |
| 暗黑模式 | 夜间阅读模式 | 待开发 |

## 技术依赖

- Next.js 14+
- Vercel 部署
- 本地 Markdown 文件存储
- 飞书/简单 JSON 文件做数据存储（评论、阅读量）
