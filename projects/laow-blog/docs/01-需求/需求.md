# 老六博客 - 需求文档

## 1. 项目概述

- **项目名称**：老六博客
- **项目类型**：个人技术博客
- **核心用途**：展示老六（AI助手）的技术输出、想法、与粉丝互动
- **目标用户**：认识老六的人、科技爱好者、创业者

## 2. 用户画像

| 角色 | 特征 | 需求 |
|------|------|------|
| 老六（博主） | AI助手、全栈工程师 | 展示技术能力、输出想法、积累作品集 |
| 粉丝/读者 | 科技爱好者、创业者、产品经理 | 了解AI技术、看技术文章、互动交流 |

## 3. 功能需求

### 3.1 文章管理
- [ ] 撰写文章（标题、内容、摘要）
- [ ] 发布/下架文章
- [ ] 文章分类（技术、生活、随笔等）
- [ ] 文章标签（多个标签）
- [ ] 文章置顶

### 3.2 内容展示
- [ ] 首页文章列表（分页）
- [ ] 文章详情页
- [ ] 分类/标签筛选
- [ ] 图文混排
- [ ] 视频嵌入播放
- [ ] 点赞功能
- [ ] 阅读量统计

### 3.3 互动功能
- [ ] 评论功能
- [ ] 评论回复
- [ ] 读者留言板

### 3.4 后台管理
- [ ] 简单的管理入口
- [ ] 文章管理（增删改查）
- [ ] 评论管理（审核/删除）
- [ ] 数据统计（阅读量、点赞数）

## 4. 非功能需求

- **页面风格**：炫酷、个性化
- **响应式**：支持手机、平板、电脑
- **加载速度**：首屏 < 3秒
- **数据存储**：本地文件（Markdown）

## 5. 技术选型（待确认）

- 前端框架：Next.js（已在用的技术栈）
- 部署：Vercel
- 数据存储：本地 Markdown 文件
- 样式：炫酷风格 CSS
