# 老六博客 - 接口设计

## 1. 页面接口（静态生成）

| 路径 | 方法 | 说明 |
|------|------|------|
| `/` | GET | 首页文章列表 |
| `/posts/[slug]` | GET | 文章详情 |
| `/category/[name]` | GET | 分类页 |
| `/tag/[name]` | GET | 标签页 |
| `/about` | GET | 关于页面 |

## 2. API 接口（动态）

| 路径 | 方法 | 说明 | 认证 |
|------|------|------|------|
| `/api/posts` | GET | 获取文章列表 | - |
| `/api/posts/[slug]` | GET | 获取单篇文章 | - |
| `/api/views/[slug]` | POST | 增加阅读量 | - |
| `/api/views/[slug]` | GET | 获取阅读量 | - |
| `/api/likes/[slug]` | POST | 点赞 | - |
| `/api/likes/[slug]` | GET | 获取点赞数 | - |
| `/api/comments` | GET | 获取评论列表 | - |
| `/api/comments` | POST | 提交评论 | - |
| `/api/comments/[id]` | DELETE | 删除评论 | 需要 |
| `/api/admin/posts` | GET | 后台：文章列表 | 需要 |
| `/api/admin/posts` | POST | 后台：新建文章 | 需要 |
| `/api/admin/posts/[slug]` | PUT | 后台：更新文章 | 需要 |
| `/api/admin/posts/[slug]` | DELETE | 后台：删除文章 | 需要 |

## 3. 接口详情

### 3.1 文章列表
```json
GET /api/posts?page=1&limit=10&category=技术&tag=AI

Response:
{
  "posts": [
    {
      "slug": "hello-world",
      "title": "你好世界",
      "excerpt": "这是摘要...",
      "date": "2026-03-17",
      "category": "技术",
      "tags": ["AI", "编程"],
      "views": 100,
      "likes": 10
    }
  ],
  "total": 50,
  "page": 1,
  "limit": 10
}
```

### 3.2 阅读量
```json
POST /api/views/hello-world

Response: { "views": 101 }
```

### 3.3 点赞
```json
POST /api/likes/hello-world

Response: { "likes": 11 }
```

### 3.4 评论
```json
GET /api/comments?slug=hello-world

Response:
{
  "comments": [
    {
      "id": "c1",
      "slug": "hello-world",
      "author": "粉丝A",
      "content": "写得真好！",
      "date": "2026-03-17T10:00:00Z"
    }
  ]
}

POST /api/comments
Body: { "slug": "hello-world", "author": "粉丝A", "content": "写得真好！" }

Response: { "id": "c2", "success": true }
```

## 4. 认证方式

- 后台管理：简单密码（环境变量 ADMIN_PASSWORD）
- 读者互动：无需认证
