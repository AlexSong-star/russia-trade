# 老六博客 - 数据结构设计

## 1. 文章存储（Markdown）

路径：`posts/*.md`

```yaml
---
title: 文章标题
date: 2026-03-17
category: 技术
tags: [AI, 编程, 老六]
excerpt: 文章摘要...
published: true
top: false
---

# 正文内容

支持 Markdown 语法...
```

| 字段 | 类型 | 说明 |
|------|------|------|
| title | string | 文章标题 |
| date | string | 发布日期 YYYY-MM-DD |
| category | string | 分类（单选） |
| tags | array | 标签（多选） |
| excerpt | string | 摘要 |
| published | boolean | 是否发布 |
| top | boolean | 是否置顶 |

## 2. 阅读量/点赞（JSON 文件）

路径：`data/likes.json`

```json
{
  "hello-world": { "views": 100, "likes": 10 },
  "second-post": { "views": 50, "likes": 5 }
}
```

## 3. 评论存储（飞书 Bitable）

**App**：老六博客数据  
**表**：评论表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | 自动ID | 评论ID |
| slug | 文本 | 文章标识 |
| author | 文本 | 评论者 |
| content | 文本 | 评论内容 |
| created_time | 创建时间 | 发布时间 |
| status | 单选 | pending/approved/rejected |

## 4. 分类数据（JSON）

路径：`data/categories.json`

```json
[
  { "name": "技术", "slug": "tech", "count": 10 },
  { "name": "生活", "slug": "life", "count": 5 },
  { "name": "随笔", "slug": "essay", "count": 3 }
]
```

## 5. 标签数据（JSON）

路径：`data/tags.json`

```json
[
  { "name": "AI", "slug": "ai", "count": 8 },
  { "name": "编程", "slug": "coding", "count": 6 },
  { "name": "老六", "slug": "laoliu", "count": 15 }
]
```

## 6. 文件目录结构

```
laow-blog/
├── posts/                    # Markdown 文章
│   ├── 2026-03-17-hello.md
│   └── ...
├── data/                     # 数据文件
│   ├── likes.json           # 阅读量/点赞
│   ├── categories.json      # 分类
│   └── tags.json            # 标签
└── bitable/                  # 飞书 Bitable 配置
    └── comments.json        # 评论配置
```

## 7. 数据流程

```
┌─────────────┐     ┌─────────────┐
│  Markdown  │────>│  静态生成   │
│   文章     │     │   页面      │
└─────────────┘     └─────────────┘
                           │
                           ▼
┌─────────────┐     ┌─────────────┐
│   JSON     │<───>│  API 接口   │
│  阅读/点赞  │     │   服务      │
└─────────────┘     └─────────────┘
                           │
                           ▼
                    ┌─────────────┐
                    │  飞书       │
                    │  Bitable    │
                    │   评论      │
                    └─────────────┘
```
