# 开发计划

## 阶段一开发清单

### Skill 1：russia-trade-searcher（客户搜索工具）
**功能**：从 B2B/黄页/展会网站搜索潜在客户

**开发内容**：
- [ ] B2B-Center 网站爬虫（b2b-center.ru）
- [ ] Metaprom 黄页爬虫（metaprom.ru）
- [ ] IndustryStock 黄页爬虫（industrystock.ru）
- [ ] 展会参展商爬虫（metallurgy-russia.ru、metal-expo.ru）
- [ ] Yandex 搜索集成（Playwright 浏览器自动化）
- [ ] 去重逻辑
- [ ] 输出：客户基本信息（公司名、官网、来源渠道）

---

### Skill 2：russia-trade-researcher（公司背调工具）
**功能**：采集公司详细信息

**开发内容**：
- [ ] 官网信息采集（Playwright）
  - [ ] "О компании" 页面提取
  - [ ] 主营产品提取
  - [ ] 地址提取
- [ ] Yandex Maps 地址补全
- [ ] 是否有 МНЛЗ（连铸机）判断
- [ ] 采购模式判断（集中/分厂）
- [ ] A/B/C 自动分类
- [ ] 输出：完整背调信息

---

### Skill 3：russia-trade-contact-finder（联系人深挖工具）
**功能**：深挖采购负责人联系方式（核心环节）

**开发内容**：
- [ ] 官网联系表单采集（Playwright）
  - [ ] "Отдел закупок" 页面
  - [ ] "Контакты" 页面
- [ ] Yandex 搜索联系方式
- [ ] LinkedIn 采购负责人搜索（Playwright）
- [ ] Hunter.io API 集成（查邮箱）
- [ ] Snov.io API 集成（查邮箱）
- [ ] 输出：联系人信息（姓名、职位、邮箱、电话）

---

### Skill 4：russia-trade-letter-generator（开发信生成工具）
**功能**：根据背调信息生成俄语开发信

**开发内容**：
- [ ] 俄语开发信提示词模板
- [ ] MiniMax LLM API 集成
- [ ] 单客户开发信生成
- [ ] 批量开发信生成
- [ ] 输出：俄语开发信文本
