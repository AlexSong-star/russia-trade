# 功能清单

## 阶段一：数据交付

### P0（核心功能，必须有）

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 客户搜索 | 从 B2B/黄页/展会网站搜索潜在客户 | P0 |
| 公司背调 | 收集公司规模、主营产品、产能等 | P0 |
| 客户分类 | 按 A/B/C 三类自动打标签 | P0 |
| 开发信生成 | 根据背调信息生成俄语开发信 | P0 |
| 数据汇总报告 | 生成客户清单 Excel/飞书表格 | P0 |

### P1（重要功能）

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 联系人深挖 | 找到采购负责人邮箱/电话 | P1 |
| 渠道自主拓展 | 主动发现新渠道网站 | P1 |
| 每日报告推送 | 飞书推送新客户清单 | P1 |

### P2（辅助功能）

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 背调模板优化 | 根据反馈优化背调信息颗粒度 | P2 |
| 开发信模板库 | 建立常用开发信模板 | P2 |
| 发送时间建议 | 根据莫斯科时间给出建议 | P2 |

## 阶段二：SaaS 平台

### P0（核心功能）

| 功能 | 说明 | 优先级 |
|------|------|--------|
| Web 管理后台 | 客户自助管理 | P0 |
| 自动化搜索任务 | 定时执行多渠道搜索 | P0 |
| 邮件发送系统 | SendGrid 接入 | P0 |
| Telegram Bot 发送 | 自动发送开发信 | P0 |
| WhatsApp 发送 | WhatsApp Business API | P0 |

### P1（重要功能）

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 发送状态跟踪 | 送达/打开/回复 | P1 |
| 客户管理 | 查看/筛选/更新客户 | P1 |
| 关键词配置 | 灵活配置搜索词 | P1 |
| 发送时间配置 | 自定义发送时间 | P1 |

### P2（优化功能）

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 效果分析 | 回复率/转化率统计 | P2 |
| 自动化分类优化 | 根据背调自动分类 | P2 |
| 多账号管理 | 切换发送账号 | P2 |
