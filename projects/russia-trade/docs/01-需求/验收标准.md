# 验收标准

## 阶段一验收标准

### 交付物验收

#### 1. 客户清单
- [ ] 包含公司名称、官网、邮箱、电话
- [ ] 包含完整背调信息
- [ ] 包含联系人信息（姓名、职位、邮箱、电话）
- [ ] 按 A/B/C 三类正确分类
- [ ] 数据格式：飞书多维表格

#### 2. 开发信
- [ ] 语言：俄语
- [ ] 内容针对公司背调信息定制
- [ ] 每位客户有独立的开发信
- [ ] 可直接复制使用

#### 3. 每日报告
- [ ] 通过飞书推送
- [ ] 包含新客户数量
- [ ] 包含客户分类摘要
- [ ] 每日推送一次

### 功能验收

#### 客户搜索
- [ ] 能从 B2B 投标网站获取客户数据
- [ ] 能从黄页网站获取客户数据
- [ ] 能从展会网站获取参展商数据
- [ ] 搜索关键词可灵活配置

#### 公司背调
- [ ] 能收集公司地址
- [ ] 能收集规模/产能信息
- [ ] 能判断是否有 МНЛЗ（连铸机）
- [ ] 能判断采购模式（集中/分厂）

#### 联系人深挖
- [ ] 能找到官网联系表单
- [ ] 能通过 Yandex 搜索到联系方式
- [ ] 能找到 LinkedIn 采购负责人

#### 开发信生成
- [ ] 生成俄语开发信
- [ ] 内容与公司信息相关
- [ ] 有称呼、有署名

---

## 阶段二验收标准（SaaS平台）

### 系统功能
- [ ] Web 后台可登录使用
- [ ] 搜索任务可定时自动执行
- [ ] 邮件可自动发送（SendGrid）
- [ ] Telegram Bot 可自动发送
- [ ] WhatsApp 可自动发送
- [ ] 发送状态可跟踪（送达/打开/回复）

### 数据管理
- [ ] 客户数据可查看/筛选
- [ ] 客户数据可更新
- [ ] 开发状态可追踪
- [ ] 关键词可配置

### 稳定性
- [ ] 系统可长期运行
- [ ] 数据不丢失
- [ ] 发送不漏发

---

## 质量标准

### 数据质量
- 客户信息准确率 > 95%
- 邮箱有效率 > 80%
- 分类准确率 > 90%

### 开发信质量
- 俄语语法正确
- 内容专业
- 有针对性
